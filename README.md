# 🌍 Countries Explorer

A responsive web application built using HTML, CSS, and JavaScript that fetches real-time country data from the REST Countries API.

## 🚀 Features

- Display countries dynamically using API data
- Search countries by name
- Show country flags
- Display population information
- Loading indicator while fetching data
- Error handling for failed API requests
- Responsive design for desktop and mobile devices
- Interactive hover effects

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- REST Countries API

- Working with asynchronous functions (async/await)
- Dynamic DOM manipulation
